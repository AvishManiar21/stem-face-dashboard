Getting Started

Welcome to the STEM Tutor Scheduling System! Here's how to get started:

  1. Configure System Settings: Visit the Settings page to enable/disable features
  2. Add Courses: Create courses that tutors can teach
  3. Create Tutor Accounts: Add tutors to the system
  4. Set Availability: Tutors can set their available time slots
  5. Start Scheduling: Students can book appointments with available tutors
