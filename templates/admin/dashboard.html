Getting Started

Welcome to the STEM Tutor Scheduling System! Here's how to get started:

  1. Add Courses: Create courses that tutors can teach
  2. Create Tutor Accounts: Add tutors to the system
  3. Set Availability: Tutors can set their available time slots
  4. Start Scheduling: Students can book appointments with available tutors
